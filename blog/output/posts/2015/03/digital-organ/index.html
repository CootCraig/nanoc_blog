macetech.com Centipede Shield provides 64 general purpose I/O

The Centipede Shield is an add-on PCB for standard layout (Uno, Duemilanove, Diecimila) Arduino microcontroller boards. It uses the Wire I2C interface on analog pins 4 and 5 to provide 64 general purpose I/O pins. Any pin can be configured for input or output. The shield uses four Microchip MCP23017 16-pin digital I/O expander chips.
