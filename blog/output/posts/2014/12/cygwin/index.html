cygwin sshd

Add to System Path
        ;c:\cygwin\bin
        
Add a user and add to administrator group:
        username: sshdadmin
        password: XXXXXX
        
Sync cygwin passwd,group with local windows
        mkpasswd --local > /etc/passwd
        mkgroup --local > /etc/group
