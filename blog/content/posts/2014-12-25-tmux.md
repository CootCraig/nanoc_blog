First use is to run weechat all the time.

## What

~~~~~~~~~~~~
Start tmux named sessions
tmux new -s weechat weechat
tmux new -s mocp mocp
~~~~~~~~~~~~

~~~~~~~~~~~~
Attach to named sessions
tmux attach-s -t weechat
tmux attach-s -t mocp
~~~~~~~~~~~~
